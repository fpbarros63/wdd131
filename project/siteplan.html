Dev Journey Portfolio

WDD 131 — Website Planning Document (W05)

Site Name

Dev Journey Portfolio — This name plays on the idea that the website is not only a portfolio, but also a measure of my learning progress and development in web development.

Optional domain idea (not required): devjourney-portfolio.com

Site Purpose

The purpose of the website is to display a personal web development portfolio, along with the learning aspect. The website will contain information regarding my professional life (About), my projects (Projects), and how visitors can reach me (Contact).

Scenarios

Questions a visitor might ask:

  • Which projects best represent your current web development skills?
  • What technologies (HTML/CSS/JavaScript) have you practiced and where can I see examples?
  • How can I contact you for collaboration, feedback, or opportunities?

Color Schema

The site will use a clean, professional palette with strong contrast for readability. The colors below are used in this site plan document.

Primary (Navy): #0B1F3A

Used for headings, header background, and key UI emphasis.

Accent (Teal): #1FB6A6

Used for highlights, links, buttons, borders, and small accents.

Background (Off-white): #F7F8FA

Used as the main page background for a bright, modern feel.

Body Text (Near-black): #111827

Used for paragraph text to ensure legibility.

Typography

The site will use a single font family to keep the design consistent and modern.

  • Roboto — Used for headings and body text.
    • Headings: Roboto 400/700
    • Body: Roboto 100/400
